# WAIC 2026 参展资料包（首轮）

抓取日期：2026-07-21（Asia/Shanghai）

本资料包用于后续整理 WAIC 2026 各参展厂商/单位的新品、亮点和“不可错过”内容。当前是首轮资料底稿，不是最终全量厂商报告。

## 目录

- `01_原始资料/`：33 个网页或模型卡原始文件、本届展商名录原图 11 张。
- `02_摘要/首轮调研摘要.md`：大会趋势、重点产品和证据强弱的初步摘要。
- `02_摘要/来源目录.md`：原始资料与来源 URL 的一一对应清单。
- `WAIC2026_整理大纲.md`：后续正式整理建议采用的大纲。

## 资料口径

- A 级：大会、政府、主办方、厂商、产品官网等一手来源。
- B 级：新华社、央广网、第一财经等权威媒体或现场报道。
- C 级：垂直媒体、转载或厂商供稿；只用于补充线索，不单独支撑关键结论。
- 厂商自报的“全球首个”“领先”“效率提升”等表述，均按“厂商宣称”处理；只有出现独立测试或明确落地证据时，才升级为验证性结论。
- 大会规模公开口径存在“3000 余项/4000 余款展品”差异。后续正文以政府新闻发布会披露的“3000 余项技术产品”为主，另在脚注说明展商名录转载页的不同口径。

## 当前覆盖边界

大会有 1100 余家参展企业，首轮不会逐家写成完整卡片。当前做法是：

1. 保存完整展商名录原图，确保后续可追溯。
2. 先覆盖官方“十大镇馆之宝”和六条主赛道的代表厂商。
3. 对有明确新品、现场演示、奖项或量产/试点证据的单位优先形成条目。
4. 最终报告另附“全展商索引”，把未展开单位与所属赛道、展位、公开资料状态列明。
